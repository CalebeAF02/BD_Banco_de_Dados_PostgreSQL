# BD Banco de Dados PostgreSQL

Repositório dedicado aos estudos, exercícios e projetos desenvolvidos na disciplina de Banco de Dados utilizando PostgreSQL.

## Informações

- Aluno: Calebe Alves Freitas
- Universidade: Universidade de Brasília (UnB)
- Área: Banco de Dados
- SGBD utilizado: PostgreSQL

## Objetivo

- praticar modelagem de banco de dados;
- desenvolver consultas SQL;
- aprender criação e manipulação de tabelas;
- praticar relacionamentos entre entidades;
- estudar normalização;
- desenvolver projetos utilizando PostgreSQL;
- praticar organização de projetos acadêmicos no GitHub.

## Conteúdo

O repositório contém atividades relacionadas a:

- comandos SQL;
- `SELECT`, `INSERT`, `UPDATE` e `DELETE`;
- criação de tabelas;
- chaves primárias e estrangeiras;
- relacionamentos;
- modelagem entidade-relacionamento;
- normalização;
- consultas avançadas;
- projetos acadêmicos.

## Tecnologias

- PostgreSQL
- SQL
- pgAdmin

## Objetivo acadêmico

Este repositório tem como finalidade registrar a evolução prática na disciplina de Banco de Dados, organizando exercícios, projetos e estudos realizados durante a graduação.
