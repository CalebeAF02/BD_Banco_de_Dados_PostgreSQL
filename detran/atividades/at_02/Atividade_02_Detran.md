# Atividade 02 — Criar BD do DETRAN (MySQL)

## Enunciado

Crie o BD do DETRAN utilizando o SGBD MySQL.

- Usuário: `aluno`
- Senha: `aluno`

- Comando no LINUX MySQL: `mysql -u aluno -p`

- \r banco (para conectar com o BD que está na máquina)

## Tarefas (comandos DML para testar)

1. Inserir 2 ou 3 registros em cada tabela do banco.
2. Atualizar pelo menos 5 registros em quaisquer tabelas do banco.
3. Apagar pelo menos 5 registros em quaisquer tabelas do banco.

## Observações

- Garanta que o banco `detran` (ou o nome escolhido) exista antes de executar os comandos.
- Use `USE detran;` para selecionar o banco dentro do cliente MySQL.
- Registre aqui os comandos SQL utilizados (INSERT, UPDATE, DELETE) e os resultados observados.
